# Social Media Analytics

This project is adapted from the EV Charging Predictions project.
It uses Apache Spark and RandomForestRegressor to predict social media engagement.

## How to Run in Codespaces

1.  **Run Setup Script** (Installs Java & Dependencies):
    ```bash
    bash codespace_setup.sh
    ```

2.  **Generate Data** (Optional, if you want fresh data):
    ```bash
    python generate_data.py
    ```

3.  **Run the Application**:
    ```bash
    python -m streamlit run streamlit_app.py
    ```
